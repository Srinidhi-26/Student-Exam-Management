# Student-Exam-Management

URL: http://127.0.0.1:5000/ For Home Page.

There are 4 Button on Home Page Create Student, Delete student, Student list and Student Marks.

Project Uses Methods POST and GET and it is based on CRUD (CREATE, READ, UPDATE, DELETE) Data.

Search Bar for searching the students is also done in this project. 

Python Flask Framework, HTML, CSS, and sqlAlchemy used in this project.
